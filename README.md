## 🎯 Acerca del Proyecto

**BOOST WELL NUTRITION** es una plataforma de comercio electrónico completa, diseñada para ofrecer productos saludables 100% naturales. Desarrollada como proyecto académico para la asignatura de **Desarrollo de Aplic. Web con Soft. Interpret. en el Cliente**, implementa las mejores prácticas de desarrollo web moderno.
